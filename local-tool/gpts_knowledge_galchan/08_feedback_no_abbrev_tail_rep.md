# ガルch アルファベット略語禁止＋語尾連続/重複禁止

ガルちゃんyt台本では:
- **アルファベット略語（PB/NB/OEM/PSE/QC/HC/DS/SC/GMS/EC等）を禁止**。必ずカタカナ/フル日本語で書く（PB→プライベートブランド）
- **「〜の。」連続2行まで**（3行目で違う語尾必須）
- **同一語尾3行連続禁止**（〜だよ/〜よね/〜の/〜だ等）
- **「〜の。」全体出現率20%以下**（WARN）

**Why:**
- VOICEVOX読み上げで「PB」が「ピービー」と読まれて視聴者に意味不明
- 同一語尾連発・「〜の。」連発で読み上げが単調・後半失速
- 2026-04-28 自ガル12（ドンキ）台本でユーザー指摘「PB略禁止・語尾重複多すぎ」
- ユーザー要求「今後ミスしないように実行構築」

**How to apply:**
- 台本生成時（galchan-script スキル含む）の生成プロンプトで明示禁止
- レビュー（galchan-review）の検証項目に追加
- 保存前 pre_save_gate.mjs で機械検出（FAIL/WARN）
- DB/rules/台本ルール.md に2セクション追加（句読点ルール直後）

**禁止略語と置換例（必ずフル日本語化）:**
| NG略語 | 置換 |
|---|---|
| PB | プライベートブランド |
| NB | ナショナルブランド |
| OEM | 他社ブランド供給品 |
| ODM | 設計まで他社委託品 |
| PSE | 電気用品安全法マーク |
| QC | 品質管理 |
| HC（文中の略） | ホームセンター |
| DS | ドラッグストア |
| SC（モール） | ショッピングセンター |
| GMS | 総合スーパー |
| EC（通販） | 通販/ネット通販 |

**例外（読み上げOK）:**
- 商品名固有: KFC、JR、NTT、KDDI 等
- 単位: kg/cm/g/ml/L 等

**機械チェック（pre_save_gate.mjs）:**
- TSV 2列目に `\bPB\b` 等含まれる → FAIL
- TSV 2列目末尾を全行スキャン
  - 「〜の。」終わりが連続3行以上 → FAIL
  - 同一末尾2文字が連続3行以上 → FAIL
  - 「〜の。」終わりが全本文行の20%超 → WARN

**修正方針:**
- 「〜の。」を別語尾に置換: 「〜のよ。」→「〜なんだよ。」「〜だしね。」「〜って思った。」「〜なんだよね。」
- スレ民ごとのキャラに合った語尾を使い分け（スレ民1=驚き系/スレ民3=知識系/スレ民6=日常系）
