# 台本本文「ガル民」呼称＋引用元明示の二重禁止

**ルール:**
1. ガル台本本文で**「ガル民」を主語・呼称・引用元として使うのを全面禁止**

3. 「他にもそういう人いる」を表現したい時は**主語ぼかし型**（「って言う人も多いよね」「って人いるよね」「うちも〜」「私も〜」）で書く

## 🚨 追加ルール（2026-04-28 v2修正で再発した違反受け）

### F. 主語ぼかし型フレーズの連発禁止

ガル民/出典明示型を主語ぼかし型に書き換える際、修正AI（自分含む）が「よく聞く話よね」「あるあるだよね」等を**連発**してしまう事故が発生（自ガル12 v2修正で4箇所連発・ユーザー指摘 #6 Discord 1498816368470921449「よく聞く話ね。連発もだめ。」）。

| パターン | 上限 |
|---|---|
| 「よく聞く話／よく聞く」 | **1回まで** |
| 「あるあるだよね／あるある」 | **1回まで** |
| 「うちもそう／私もそう」 | **2回まで** |
| 「〜って人いるよね／〜って人多い」 | **2回まで** |
| 「私の周りでも／私の周りに」 | **1回まで** |

**分散ストック（順番に使い回す）:**
1. 「うちもそう」「私もそう」
2. 「あるあるだよね」
3. 「私の周りでもいた」
4. 「同じ目に遭ったことある」
5. 「結構いる」「結構ある」
6. 「〜って人多いと思う」
7. 「ママ友でも〜って話出る」
8. 「うちの姉も〜」「うちの母も〜」

### G. 〜よ。／〜よね。／〜わよね。 現代少数派・比率制限

ユーザー指摘 #5（Discord 1498816194231275733）「〜よ。よね。わよね。とかは現代でいうひとすくないからさ比率すくなめにして」。

| 末尾パターン | 全本文行に対する上限比率 |
|---|---|
| 〜よ。 / 〜よね。 / 〜わよね。 / 〜わ。 / 〜だわ。 合計 | **30%以下** |
| 同一末尾2文字（〜よ・〜ね・〜わ等）連続 | **2行まで**（既存ルール） |

**現代風代替:**
- 〜よ。 → 〜だよ。 / 〜なんだ。 / 〜って思う。 / 〜って。
- 〜のよ。 → 〜なんだ。 / 〜だよね。 / 〜って話。
- 〜だわ。 → 〜だ。 / 〜なんだ。
- 〜わよね。 → 〜じゃない？ / 〜だよね。

### H. ナレーション1文1行ルール（次回からの予防）

ユーザー指摘 #7（Discord 1498815987657609447）「ナレーション1文1行なのに冒頭2文1行になってるとこあるから次回から気をつけて」。

- ナレーション話者の行は**1文1行厳守**
- 句点「。」を含む2文以上を1行にまとめない
- 例外: 「皆さんこんにちは！今回は、」のような自然な続き挨拶のみ
