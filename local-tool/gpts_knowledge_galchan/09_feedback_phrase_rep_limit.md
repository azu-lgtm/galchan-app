# ガル台本重複フレーズ上限ルール

**ルール**: 同じ言い回し・同じ証言パターンが繰り返されると「作り物」感が出て後半失速する。1台本あたりの重複フレーズに上限を設けて分散・言い換え必須。

**Why**:
- 自ガル11 v8で他AIから「スレ民6の元店員言及20回超」「SNS/ママ友証言10回」「代わりに何買えばいいの？が8回ほぼ同形」「値段は2倍だけど結局安いが10回超」と指摘
- 自ガル11 v11後にもユーザーから「本当に〜10箇所超」「今夜確認する7箇所」「家族の命/家族を守る6箇所」と過剰指摘（11:36追加指摘）
- 後半6位以降で「またか」と作り物臭が出る・40代女性視聴者には説教感が出る

**How to apply（上限・違反時はpre_save_gate FAIL）**:
| パターン | 上限 |
|---|---|
| 「売り場にいた頃/仕入れ担当の頃/20年働いた」 | 6回 |
| 「SNSでも/インスタでも」 | 2回 |
| 「ママ友（の間）でも/知り合いでも/同僚でも」 | 2回 |
| 「家族にはすすめない/家族に絶対使わせない」 | 3回 |
| 「家族の命を守る最小投資/医療費より安い」 | 3回 |
| 「値段は2倍/3倍だけど結局安い」 | 3回 |
| 「代わりに何買えばいいの？」同形 | 2回（残りはバリエーション化・WARN） |
| 🆕「本当に〜」系（怖/油断/危ない/助かる/泣ける/鳥肌/これだけは/やめにしよう） | **5-6回** |
| 🆕「今夜〜確認/見/チェック」「今日帰ったら〜」系 | **3回**（サーキュレーター/LED/最後のまとめのみ） |
| 🆕「家族の命/家族を守る/命に直結」系派生 | **3回**（耐震/LED/サーキュレーターのみ） |

**バリエーション必須フレーズ**:
- ✅「代わりに何買えばいいの？」 → 「じゃあ選ぶ時はどこ見ればいい？」「買い替えるなら何を基準にする？」「安全寄りで選ぶなら？」「これ、今から買う人はどうすればいい？」
- ✅「20年働いた私が家族に絶対すすめない」 → 「自分の家には別のを置いてた」「うちは違うメーカー使ってる」
- 🆕 「本当に怖い」 → 「ゾッとする/胃が冷える/血の気が引く/震えた」
- 🆕 「今夜確認する」 → 「気になる人は早めにチェック/週末まとめて見直し」（3箇所のみ）
- 🆕 「家族の命を守る」 → 「家族の暮らしを支える/家族を不安にさせない」（耐震/LED/サーキュレーターの3箇所のみ）

**🆕 素材断定NG（2026-04-25 11:36ユーザー指摘・C4補強）**:
- ❌ 国産杉/ヒノキ「反らない/曲がらない」断定 → ✅「反りにくい/ロットによっては反ることもあるが少ない」
- ❌ 「絶対大丈夫/絶対安全/絶対失敗しない」 → ✅「失敗しにくい/安心して使える」
- ❌ 「ビクともしない」 → ✅「揺るぎなく使える/長く使える」

**残してOKの重複（視聴者の心の声）**:
- 「うちにもある」「知らなかった」「代わりに何選べばいい？」 → 共感と不安の引っ張り役で必要
- ただし同じ話者が繰り返さない（別のスレ民で言わせる）
