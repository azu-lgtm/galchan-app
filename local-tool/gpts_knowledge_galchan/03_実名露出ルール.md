# 公的機関措置命令ベース実名露出ルール（ガル台本・名誉毀損リスク回避）

> **ガル台本本文・サムネ・タイトル・概要欄での特定商品名の実名出しは、公的機関（消費者庁・厚労省・公取委・農水省等）が措置命令・課徴金・名指し警告を出した案件に限定する。**
> **それ以外で実名+「危険」「ヤバい」断定すると、名誉毀損訴訟リスク・アンチコメ大量発生リスクが高い。**

---

## 1. 背景

### 競合（ガル姫「消費者庁が名指しで警告」）の文字起こし分析

競合は実名バリバリ出してるが、すべて消費者庁が措置命令を出した案件のみ:

| 商品 | 措置命令理由 |
|---|---|
| ファブリーズお風呂消臭剤（P&G） | 景表法違反（合理的根拠なし） |
| ユーフィー ロボット掃除機（Anker） | 発火事故 |
| テモリラボ（ST社） | 花粉ガード根拠なし |
| タイガー魔法瓶 電気ケトル | こぼれない表示が虚偽 |
| ジャパネット高田 おせち | 二重価格表示 |
| アノブ美容クリーム | 「シミ完全消滅」根拠なし |
| 大正製薬 アンチエイジングサプリ | ステマ |
| クレベリン（大幸薬品） | 効果根拠なし |
| ジュエルアップ・モテアジュ | バストアップ根拠なし |
| チョコザップ・ライザップ | ステマ・24時間表示違反 |
| 味の素 冷凍宅配弁当 | ステマ |

### 競合の防衛ロジック（パクれる）

1. 「消費者庁が措置命令を出した」事実だけ読む → ナレーターが断定しない
2. コメント主が体験を話す → 個人の感想（口コミは表現の自由）
3. 「今後注意して」で締める → 商品否定ではなく注意喚起

## 2. 実名OK条件（5要件すべて満たす場合のみ）

### A. 公的機関の措置命令あり

✅ 消費者庁・厚労省・公正取引委員会・農林水産省・経済産業省等の公的機関が以下のいずれかを出している:
- 措置命令
- 課徴金納付命令
- 名指しの警告・公表
- リコール命令
- 営業停止命令

### B. 措置命令の理由が公表されている

✅ 以下のいずれかの違反根拠が公表記事・公的機関プレスリリース等で確認できる:
- 景品表示法違反（不当表示・優良誤認・有利誤認）
- 薬機法違反
- ステマ規制違反（2026年施行）
- 特定商取引法違反
- 食品衛生法違反

### C. 引用は事実報道としてのみ

✅ 「○○庁が措置命令を出した」「課徴金○○円が命じられた」「景表法違反と認定された」等の**事実引用フレーズ**を使う。

### D. 断定表現NG

✅ ナレーター・スレ民が「危険」「ヤバい」「殺された」「絶対やめろ」等の断定表現を使わない。

### E. 締めは「注意して」レベル

✅ 結論は「注意して」「慎重に検討して」「個人の感想として参考に」レベルで止める。「絶対これ食べるな」「死ぬよ」型はNG。

---

## 3. 実名NG条件（1つでも該当したらNG）

### A. 訴訟係争中

❌ 小林製薬「紅麹」死亡訴訟係争中等、最終確定していない案件は実名NG。

### B. 個人ブランド・薬機法グレー商品

❌ 措置命令未確定の個人ブランド商品・薬機法グレーゾーンの商品は実名NG。

### C. 断定表現と実名の組合せ

❌ 「○○は危険」「○○は詐欺」「○○は致命的」「○○で死ぬ」等、実名+断定の組み合わせは全面NG。

### D. ナレーター/スレ民独自盛り

❌ 公的機関発表を超えてナレーター・スレ民が独自に「マジでヤバい」「本気で危険」等の評価を盛り込むのはNG。

### E. 命令ネガとの組合せ

❌ 「絶対これ食べるな」「絶対これやめろ」「絶対これ買うな」型の命令ネガをサムネ・タイトル・本文で実名と組み合わせるのはNG。

## 4. OK表現 vs NG表現対比表

| 場面 | OK（公的機関事実報道） | NG（名誉毀損リスク） |
|---|---|---|
| 商品名+違反根拠 | 「消費者庁が課徴金1,086万円を命じた」 | 「○○は危険」 |
| 商品名+違反種別 | 「景表法違反で措置命令を受けた」 | 「○○は詐欺商品」 |
| ステマ違反 | 「ステマ規制違反でST社が措置命令」 | 「○○は絶対やめろ」 |
| 体験談 | 「ガル民が『やめたら痩せた』と告白」 | 「○○で死人が出た」 |
| 効果なし体験 | 「個人の感想として効果を感じなかった」 | 「○○は効かない（断定）」 |
| 締め | 「気になる人は注意して」 | 「絶対これ食べるな」 |

## 7. 違反検出（pre_save_gate.mjs ガード24）

### 機械検出パターン

```regex
# 実名+断定表現セット（5字以内に NG断定語が出現）
[ぁ-んァ-ヶ一-龥A-Za-z0-9・]{2,12}(は|が|って).{0,15}(危険|致命的|殺|死ぬ|詐欺|絶対やめろ|絶対これ食べるな)

# 命令ネガ
絶対これ食べるな|絶対これ飲むな|絶対これやめろ|絶対これ買うな

# 永久ブラックリスト（紅麹は無条件FAIL）
紅麹
小林製薬.*紅麹
紅麹.*(死|被害|致命|殺|事故)

# ナレーター/スレ民独自盛り（公的機関情報以外で「ヤバい」断定）
.*(は|が|って).*(マジでヤバい|本当にヤバい|本気でヤバい|めっちゃ危険)
```

いずれか1件でもヒット → **pre_save_gate FAIL（書き直し必須）**

## 8. 台本作成時の事前チェックリスト

台本本文に実名（商品名・ブランド名）を出す前に、すべて✅か確認:

- [ ] 消費者庁・厚労省・公取委等の公的機関が措置命令を出している
- [ ] 措置命令の理由が公表記事・公的機関プレスリリース等で確認できる
- [ ] 引用は「○○庁が措置命令を出した」レベルの事実報道のみ
- [ ] ナレーター・スレ民の独自「ヤバい」「危険」断定がない
- [ ] 訴訟係争中ではない（最終確定済み）
- [ ] 締めは「注意して」レベル
- [ ] サムネ・タイトルに実名+断定の組合せがない

**1つでも未チェック → 実名出さずカテゴリ表記（例: 「機能性表示飲料」「サプリ」「ダイエット食品」）に統一**
