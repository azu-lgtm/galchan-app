# 商品紹介広告コピー＋製作者目線＋過剰演出＋業界用語 全面禁止

**ルール:** ガル台本本文で以下4パターンを全面禁止

1. **広告コピー風商品紹介**（「大手代替なら○○・Amazonで○○円・○○年ロングセラー」型）
2. **製作者・ナレーター目線の演出**（「あとで話す」「○○より上にした理由」等）
3. **過剰演出セリフ**（「気になりすぎて」「聞きたくて仕方ない」等）
4. **業界用語・マーケティング解説**（「業界の常識」「設計」等）

**Why:**
- 2026-04-29 ユーザー指摘・自ガル12 v3で **「大手代替なら○○ブランド。Amazonで○○円、○○年ロングセラー」型が12箇所連発**してた（広告コピー風で井戸端会話じゃない）
- ワーカーから「紹介が広告みたいだった」と指摘（自ガル12 v3 L172について）
- 同台本に「マンゴーグミより上にした理由はあとで話す」（製作者目線）「気になりすぎて、もう続きが聞きたくて仕方ない」（過剰演出）「目的なく入った客が一番落とす設計」（業界用語）も混入
- 競合（ガル姫/ガル姫/ガルねこ/ガルこめ）32パターン分析の結果、競合は「大手代替なら〜Amazon〜」型を一切使わず、**個人体験＋年数曖昧／やっぱりこれ／価格軽く触れる／流通入手難／ブランド分散** の5型で自然に紹介

**How to apply:**

### 1. 広告コピー風商品紹介の汎用禁止

**NG構造（1セリフに3要素以上）:**
- ブランド名 + 価格（数値直接） + 年数（数値直接）
- ブランド名 + 価格 + ロングセラー
- 「大手代替なら○○。Amazonで○○円、○○年の○○」型

**OK構造（5型・毎回違うバリエで）:**
詳細は `DB/rules/会話で商品が出る時の自然パターン集.md` を参照（galchan-script.md Step 0 で必読）

A型 個人体験＋年数曖昧:「うちは○○、もう数十年使ってる」
B型 やっぱりこれ:「結局○○に戻った」「いつもこれ。浮気できない」
C型 価格軽く触れる:「安い時にまとめ買い」「セールでお得」
D型 流通入手難:「最近店頭で見かけない」「ネットでまとめ買い」
E型 ブランド分散:価格・年数・効果を別セリフに分散

**重要:** これらの例文を毎回そのままコピペしない（毎回違う組み合わせで書く）

### 2. 製作者・ナレーター目線禁止

**NG型:**
- 「あとで話す」「○○より上にした理由」「次のお話は」「今日紹介するのは」型の構成説明・予告

**理由:** ガル民同士の井戸端会話のはずなのに、台本作家の声が漏れてる

**OK代替:** 普通に本題に入る（予告なしでストレート）

### 3. 過剰演出セリフ禁止

**NG型:**
- 「気になりすぎて、もう続きが聞きたくて仕方ない」「衝撃で言葉が出ない」「マジで今すぐ見直して」

**理由:** 芝居がかってる・素朴な反応じゃない

**OK代替:** 「え、それは怖い」「マジか」「ぞっとする」「やばっ」等の自然な驚き

### 4. 業界用語・マーケティング解説禁止

**NG型:**
- 「業界の常識」「目的なく入った客が一番落とす設計」「マーケティング戦略」「ターゲティング」型

**理由:** 主婦の井戸端会話じゃなくマーケティング解説者の声

**OK代替:** 「店ってさ、〜なんだよね」「売り場で見てた感じ、〜」等の体感的・感覚的表現

### 適用範囲
- 全ガルちゃんまとめ台本（ナレーション・スレ民全員・全シーン）
- 全スキル（galchan-script / galchan-review / galchan-final-check）
- 外注・別AI経由の台本も適用
- 自ガル1〜11の文体踏襲（広告コピー型は使ってなかった）
