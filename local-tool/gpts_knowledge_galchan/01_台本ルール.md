# 台本生成ルール（Single Source of Truth）

> **TSVフォーマット・登場人物・SE配置・禁止事項・固定文の唯一の正。**
> galchan-script / galchan-review / GPTs / 手動生成すべてがこのファイルを参照する。
> 外注完成台本（_done.tsv）のフォーマットに完全準拠。
>
> **注意:** トーン・会話の温度感は、このファイルではなく**競合文字起こし（32本）を直接参照**して吸収する。
> 構造・テンポ・感情動線は**勝ちパターン統合版 + 分析原本**を参照する。
> このファイルの役割は「フォーマットと禁止事項のルール集」である。

## TSVフォーマット（done.tsv準拠・厳守）

### 列構成
| 列 | 内容 | 備考 |
|---|---|---|
| 1列目 | 話者名 | ナレーション/タイトル/イッチ/スレ民1〜6 |
| 2列目 | 本文 | セリフ内容 |
| 3列目 | SE | 基本空欄。10行に1回程度でSE1/SE2交互 |

- ヘッダー行なし（1行目からナレーション）
- コロン（：）不要。列はタブ区切りのみ
- かぎ括弧「」は使わない

### 登場人物（完全一致必須）
| 名前 | 役割 | 備考 |
|---|---|---|
| **ナレーション** | 進行 | 冒頭とエンディングのみ。本文中には出ない |
| **タイトル** | テーマ表示 | 2行目固定。**10〜20文字程度**（2026-04-16 競合実態に合わせて改定。ガル姫27字・ガルねこ17字） |
| **イッチ** | スレ主 | 各トピックの口火。全体の10%以下 |
| **スレ民1** | 驚き・共感 | 感情的な反応、短め |
| **スレ民2** | 体験談 | 「うちも〜」「実際に〜」で始める |
| **スレ民3** | 追加知識 | 「それって〜が原因で」「実は〜」 |
| **スレ民4** | 疑問・対策 | 「じゃあどうすれば」「代わりに何を」 |
| **スレ民5** | まとめ・納得 | 「これは広めなきゃ」「勉強になった」 |
| **スレ民6** | 別視点・日常 | 「うちの旦那も〜」「子どもが〜」 |

> **禁止**: ナレーター、イチコ、その他の名前は一切使わない
> スレ民ごとに口調・温度・視点を少しずつ変える。全員が同じ文章力にならないこと。

## イントロ

> **競合の実態:** 全競合が「今回は〇〇についてまとめました。それではどうぞ」パターン。2-3文・10秒以内で本題に入る。長いイントロの動画はゼロ。
> **自チャンネルの差別化:** ガルこめパターン＋属人性（「私も〜」で消費者目線の一言）。情報提供者ではなく視聴者と同じ立場で話す。

```
ナレーション	皆さんこんにちは！今回は、
タイトル	（10〜20文字のテーマ・競合実態に合わせ拡張）
ナレーション	（体験一言）。（CTAに自然につながる一言）
ナレーション	後悔する前に、チャンネル登録していただけたら嬉しいです。
ナレーション	それではいってみよう！！
```

### イントロの構成ルール
1. **挨拶**: 「皆さんこんにちは！今回は、」← 固定（1行目）
2. **タイトル**: 10〜20文字のテーマ提示（2行目）※競合実態に合わせ2026-04-17改定
3. **体験一言＋つなぎ**（3行目）:
   - 「私も〜」の体験一言＋**CTAに自然につながる一言**
   - **「についてまとめました。」は不要**（情報提供者の視点になるため削除）
   - **「私も〜」は消費者・視聴者と同じ目線**（調べた・まとめた等の発信者視点NG）
   - ✅「私も知らずに買ってたものばかりでした。同じ失敗をしたくない方は、」
   - ✅「私も全部持ってて焦りました。知らないと損する情報をお届けしてるので、」
   - ❌「についてまとめました。私も〜」→ 情報提供者の視点。不要
   - ❌「農林水産省も注意を出している身近な食品、知らずに食べ続けてました。」→ 説明的すぎる
4. **CTA**: 「後悔する前に、チャンネル登録していただけたら嬉しいです。」← 固定（4行目）
5. **開始**: 「それではいってみよう！！」← 固定（5行目）
6. **合計5行（挨拶・タイトル・体験+つなぎ・CTA・開始）。** 体験→CTA→開始が自然な流れになること。
7. **イントロのネタとトピック1の内容を被らせない**（同じ食品で始めると既視感が出る）
8. **タイトルコール（2行目）に権威機関名は入れない**（YouTubeのタイトル・サムネのみ。競合も入れていない）
9. **タイトルに特定の機関名（農水省等）を入れず「国」で包括する**（特定機関名はインプレ停止リスクあり）
10. **タイトルコール ≒ YouTubeタイトルの核ワード含有必須**（2026-04-17新設）
    - 冒頭タイトルコール（台本2行目）は、**YouTubeタイトルから核となるキーワード2つ以上を含む**こと
    - 例: YouTubeタイトル「家にあったら今すぐ捨てて…国が名指しした危険な日用品」→ タイトルコール「ドラストで買い続けてた危ない商品」（「危ない」「商品/日用品」が核ワード）
    - NG例: YouTubeタイトル「危ない日用品」に対し「更年期の悩み」等の無関係なタイトルコール → サムネ-台本不一致で離脱率上昇
    - **レビュー時の機械チェック**: Grepで台本2行目とYouTubeタイトルの名詞共通性を検証

### イントロの禁止事項
- **ナレーションは1行1文**（2文を1行に詰めない。必ず改行で分ける）
- **年代（40代等）をイントロ・エンディングに入れない**（視聴者層が50代以降に偏っているため。S009参照）
- **6行以上のイントロ禁止**（5行で完結させる）
- ナレーション体験談を長く語らない
- ネタバレ禁止（具体的な食品名・商品名リストを言わない）
- 体験談で具体的な年齢を入れない
- ナレーションは丁寧でやさしい敬語

## エンディング

- ナレーション8〜10行
- ナレーションは**1行1文**（2文を1行に詰めない。必ず改行で分ける）

### エンディングの構成（順序厳守）
1. **前向きなまとめ**（2〜3行）:
   - **具体的な行動を入れる**（「私も毎朝食べてたものをひとつ変えた」等）
   - 「私も〜」で始めて寄り添う。一緒に行動する空気を作る
   - 完璧を求めない温度感
   - ✅「毎朝食べてたものをひとつ変えました」「知れた今日からはちょっと安心」
2. **コメント誘導**（1行。競合8/9本が実施）:
   - 具体的な問いかけ形式にする
   - **まとめ→問いかけを1セットにして自然につなげる**（唐突に質問しない）
   - ✅「知っているだけで毎日の買い物がちょっと変わるかもしれません。皆さんが一番びっくりした食品はどれでしたか？ぜひコメントで教えてください」
   - ✅「他にも気をつけたほうがいいものがあれば、ぜひコメントで教えてください」
   - ❌「コメントお願いします」→ 漠然としすぎ
   - ❌ まとめの直後にいきなり質問 → つなぎが唐突
3. **チャンネル固定文**（変更禁止・4行）:
```
ナレーション	このチャンネルでは後悔しないための失敗回避をテーマに
ナレーション	知らないと損する情報や私の体験談を交えてお話ししています。
ナレーション	少しでも参考になったら高評価・チャンネル登録していただけると嬉しいです!!
ナレーション	最後までご視聴ありがとうございました。
```
> 固定文の文言は一字一句変更しないこと
> 3行目（高評価・チャンネル登録）の読み上げ中にCTA画像（お辞儀イラスト）を表示する

### 🆕 エンディングトーンガイド（2026-04-25確定・視聴者ファン化目的）

> **目的**: ターゲット45-55歳女性が「このチャンネル好き」と感じる、終始低姿勢・寄り添い・労いトーンで視聴者をファン化させる
> **背景**: 自ガル11 v8エンディングが教訓調・上から目線で再修正指示を受けた

#### 必須3要素（毎回入れる）
1. **動画全体の小要約**（1-2行）
   - 「今日お話ししたのはどれも〜」「今日の話、〜が多かったかも」感
   - **私自身も〇〇でした型**で視聴者と同じ立場
2. **共感ポイントの言語化**（1-2行）
   - 「うちにもあるかも…」「これ、毎日使ってた…」（視聴者の心の声）
   - 家事・育児・仕事の見えない負担を具体化
   - 家族の何気ない一言・主婦同士の「わかる！」感覚
   - 最後のオチや軽い笑い要素
3. **ねぎらい締め**（1行）
   - 例: 「家事や育児、お仕事の合間に、ここまで見てくださって本当にありがとうございました」
   - 視聴者の暮らしに対する尊敬・感謝を具体的に
   - 抽象的な「ありがとう」ではなく、何の合間に見てくれたかを言及

#### トーン原則（OK ✅ / NG ❌）
- ✅ **終始低姿勢の敬語**: 「〜してくださって」「〜本当にありがとうございました」
- ✅ **匿名掲示板風の余韻**＋やさしい敬語のバランス
- ✅ **共感と軽い笑い**: 説教調でなく「あるある」「うんうん」感
- ✅ **私自身も〜型**: 視聴者と同じ立場で語る（情報提供者の立場NG）
- ❌ **断定・教訓調**: 「〜時代になりました」「〜すべきです」NG
- ❌ **決めつけ調**: 「家族の暮らしを守ってる」（視聴者の人生を勝手に定義）NG
- ❌ **専門家解説調**: 教える立場の口調NG
- ❌ **広告臭・売り込み**: 「〜こそ買うべき」NG

#### 🚨 毎回トーン同じ・内容毎回変える原則（厳守）
- ✅ **トーン**（低姿勢・寄り添い・労い・ほっこり）は毎台本同じ温度で固定
- ✅ **内容**（要約・共感ポイント・ねぎらい言葉）は**台本テーマに合わせて毎回違う表現**
- ❌ コピペ流用禁止: 食品テーマと家電テーマでエンディング文面が同じはNG
- ❌ テンプレ化禁止: テーマに合わせて表現を毎回考えること

#### NG例 vs OK例

| ケース | NG | OK |

| 教訓調 | 「家族の安心を決める時代になりました」 | 「家族の暮らしを支えてくださってるみなさん、本当にすごいなって思います」 |

## CTA（チャンネル登録・高評価誘導）

### 挿入タイミング（2箇所）

1. **冒頭CTA**（イントロ4行目 = 台本テキストとして読み上げ）
   - 文言: **「後悔する前に、チャンネル登録していただけたら嬉しいです。」**
   - 3行目（体験一言＋つなぎ）の直後、5行目（それではいってみよう！！）の直前

## 語尾バリエーションルール（VOICEVOX読み上げ対応）

> 同じ語尾が続くと読み上げが単調になる。意識的にバリエーションを入れる。

- **「〜よ」「〜よね」の合計を50%以下に抑える**
- 残り50%は以下のパターンに分散させる:
  - 〜んだよね（共感）/ 〜からね（理由）/ 〜って感じ（曖昧さ）
  - 〜だった（過去・体験談）/ 〜してる（現在進行）/ 〜かも（推測）
  - 〜んだけど（余韻）/ 〜だって（伝聞）/ 〜なんだよ（強調）
  - 〜だし（並列）/ 〜てほしい（願望）/ 〜ってこと（まとめ）/ 〜わ（女性的）
- **同じ語尾パターンが3行連続しない**
- スレ民ごとのキャラに合った語尾を使う（スレ民1=驚き系、スレ民3=知識系、スレ民6=日常系 等）

## 句読点ルール（VOICEVOX読み上げ対応・必須）

> 台本はVOICEVOXで読み上げるため、句読点の有無が音声の自然さに直結する。

- **1文ごとに適切な句読点（、。）を入れる** — 句読点なしの長文は読み上げが不自然になる
- **読点（、）**: 接続助詞（〜けど、〜から、〜ので）の後など、自然な切れ目に入れる。入れすぎない
  - ✅「私もこれずっと使ってたんだけど、全然効果なかった。」
  - ❌「私もこれ、ずっと使ってたんだけど、全然効果なかった。」→ 読点多すぎ。ぶつ切りに聞こえる
  - ❌「私もこれずっと使ってたんだけど全然効果なかった」→ 句点なし。次の文とつながる
- **句点（。）**: 文末に必ず入れる。体言止めでも入れる
- **感嘆符（！）・疑問符（？）**: 句点の代わりに使ってOK。ただし連続2文はNG

## 🆕 C14. 固定コメントリード1文ルール（2026-04-27確定・自ガル12 v1初版違反受け）

> **固定コメントの冒頭リード文は1文のみ。台本内のエンディング「コメント誘導」と同内容にする。**

### 推奨形式（自ガル12確定版）
```
皆さんの「これは確認するようにした」工夫があれば、ぜひコメントで教えてください。

▼皆さんに聞きたいこと
・...
```

### NG形式（自ガル12 v1初版で発生）
```
私も台本を作りながら家のドンキ買い物カゴを思い返したら、深夜のなんとなく買いで家族の食卓や家計を地味に削っていたことに気づいて、ぞっとしました。
特に焼き貝ひもとマンゴーグミは公式が回収を発表していて、家族の口に入る前に確認できて本当に良かったと思っています。

▼皆さんに聞きたいこと
```

### 理由
1. **視聴者の集中を切らさない** — 長文リードは「読み飛ばし」を誘発し、コメント率が下がる
2. **台本との一貫性** — エンディングのコメント誘導と同内容にすることで、視聴後の流れが自然
3. **シンプル誘導が有効** — 1文の方がアクション率（コメント投稿）が高い傾向

### 機械チェック方針
- 固定コメント冒頭の▼マーカー前テキストが2行を超える場合はFAIL
- 「私も台本を作りながら」「ぞっとしました」「家計を地味に削って」等のテンプレ違反フレーズを検出
