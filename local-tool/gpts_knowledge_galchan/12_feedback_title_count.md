# ガルタイトル「〇〇選」と商品リスト件数完全一致

**ルール**: YouTubeタイトルに「〇〇選」と数字を入れた場合、商品リスト件数・概要欄件数・固定コメント件数の4箇所すべてで完全一致させる。

**Why**: 自ガル11 v8で「絶対買うな10選と代わりに買うべき商品14選」とタイトルにあるのに商品リスト19件で乖離。視聴者が後半でカウントすると整合性に気付き信頼低下。他AIから「14選」のズレを指摘された。

**How to apply**:
- 台本生成時、タイトルの数字 = 商品リスト件数を最初に確定
- 19品アフィを温存したい場合は、タイトルを「19選」に揃える（ユーザー判断）
- 14品に絞るなら5品削除する（アフィ機会5件喪失）
- 「神商品まとめ」「代わりに選びたい候補」等のぼかし表現で数字回避も可
- pre_save_gate.mjs にタイトル数字とポジ商品リスト件数の機械チェック追加済み
- 4箇所一致必須: タイトル / 商品リスト / 概要欄記載件数 / 固定コメント記載件数

**自ガル11確定（2026-04-25）**: タイトル「14選」→「19選」に変更（ユーザー判断・商品リスト19件に合わせる）。サムネ数字なしで影響ゼロ。

**機械チェック例**:
```js
const titleNumMatch = title.match(/(\d+)\s*選/);
const titleNum = parseInt(titleNumMatch[1]);
if (posProducts.length !== titleNum) FAIL;
```
