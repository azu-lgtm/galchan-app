## 1. 全体シーン構成

```
[イントロ]
  ナレーション「皆さんこんにちは、今回は〜」
  タイトル（タイトルコール読み上げ）
  ナレーション（体験談）
  ナレーション「それではいってみよう！！」

[シーン転換]
  画面切り替えエフェクト（画面切り替え.PNG / 63フレーム）
  シーン切替SE（40%）

[本編]  ← ここから紫タイトル吹き出し（TextItem）表示開始
  イッチ・スレ民のセリフ繰り返し
  各セリフにいらすとや/なのなの/商品画像を配置

[エンディング]  ← ここでTextItem表示終了
  ナレーション（まとめ・チャンネル誘導）
  背景：動画素材・冒頭オープニングとテキストボックス.mp4 に切替
```

**想定動画尺**: 20〜25分

## 5. TextItem（ピンクタイトル吹き出し）

> **原則**: 位置・見た目は完全固定。台本によって変わるのは Text（タイトル文字列）・Frame・Length のみ。

### 基本設定（テンプレート実測値・固定値）
| 項目 | 固定/可変 | 値 |
|---|---|---|
| テキスト内容 | **可変** | TSVの「タイトル」行の本文 |
| 表示開始フレーム | **可変** | 本編スタートフレーム（イントロ音声尺から計算） |
| 表示長さ（Length） | **可変** | エンディングナレーション開始直前フレームまで |
| X位置 | 固定 | -950.0 |
| Y位置 | 固定 | -453.5 |
| フォント | 固定 | けいふぉんと |
| フォントサイズ | 固定 | 45.5177465392556 |
| 文字色 | 固定 | `#FF000000`（黒） |
| Layer | 固定 | 2 |

### VideoEffects（テンプレートから複製）
| エフェクト | 主要パラメータ |
|---|---|
| OutlineEffect | StrokeThickness=8.0, Blur=0.0, Quality=64.0 |
| FillBackgroundEffect | Color=`#FFE7A8CD`（ピンク）, Top=93.7, Bottom=62.4, Left=800, Right=800 |
| MaskEffect | IsEnabled=False（1つ目）, IsEnabled=True（2つ目）|
| CenterPointEffect | Horizontal=Left, Vertical=Origin |

> ※ TextItemをテンプレートから `deepcopy` して `Text`・`Frame`・`Length` だけ差し替えれば他はそのまま使える

## 7. 画像素材

> **原則**: Layer・X・Y・Zoom は固定値。台本によって変わるのは Frame・Length・ファイルパス（内容依存）のみ。

### いらすとや
- 上限: **1動画20点以内**
- Layer: 6（固定）
- X: 630 / Y: 270 / Zoom: 52%（固定）
- Frame/Length: **可変**（対応するVoiceItemと同フレーム・同尺）
- 同キーワードでも毎回違う画像を使用
- URL: https://www.irasutoya.com/

### なのなの様（イラストAC）
- 作家ページ: https://www.ac-illust.com/main/profile.php?id=oeNk1IN3
- Layer: 7
- DL制限に達した場合のみ代替可（内容・テイスト・雰囲気が合うもの）
- 自動DLは不安定なためHTMLリンク集で手動対応（現状）
- **今後の方針**: ローカルフォルダに事前DL → そこから自動参照

### 商品画像
- 台本内に商品名が出た場合、商品リストシートのリンクから画像取得
- Layer: 7〜8（固定）
- X: 659 / Y: 250 / Zoom: 100%（固定・商品は原寸に近い）
- Frame/Length: **可変**（対応するVoiceItemと同フレーム・同尺）

### 配置ルール
- 1セリフにつき1〜3枚
- テキストより上のLayer（文字が隠れないことを最優先）
- セリフと同フレーム・同尺で配置

## 8. アニメーション設定

使用エフェクト（イラストごとにローテーション、外注版ymmp実測値）:

| # | エフェクト | $type（完全修飾名の末尾） | 主要パラメータ（実装予定値） |
|---|---|---|---|
| 0 | 反復拡大縮小 | `RepeatZoomEffect` | Zoom=95.0（実装済み） |
| 1 | 反復回転 | `RepeatRotateEffect` | Rotate=5.0 |
| 2 | 反復移動 | `RepeatMoveEffect` | X=10.0, Y=0.0 |
| 3 | 跳ねる | `JumpEffect` | Height=20.0 |

ローテーション方式: `i % 4` でインデックスを循環（i=画像の通し番号）

**禁止**: 同一パターンの多用（= ローテーションせず同じものを使い続けること）

外注版で確認された追加エフェクト（将来対応候補）:
- `RepeatSkewEffect`（傾き）
- `RandomZoomEffect`（ランダムズーム）
- `ChromaKeyEffect2`（商品画像の背景抜き）

## 11. TSV→ymmp 変換ルール

### TSVフォーマット（5列タブ区切り）
```
話者\t本文\t(空欄)\tSE\tキーワード
```
- 列1: 話者（ナレーション / タイトル / イッチ / スレ民1〜6）
- 列2: 本文
- 列3: 空欄（予約）
- 列4: SE列（SE1 / SE2 / 空欄）
- 列5: いらすとや検索キーワード（Geminiが生成してWebアプリ保存時に付与）

### キーワード生成フロー
1. Webアプリ「💾 全ファイルを保存」押下
2. `gemini_generate_image_keywords()` が各セリフのキーワードを一括生成
3. TSVの列5にキーワードを追記してDriveにアップロード
4. galchan_auto.py が列5を読み取り → いらすとや自動DL

### 話者→VoiceLayer変換テーブル

| TSV話者 | VoiceLayer | X | Y | 備考 |
|---|---|---|---|---|
| ナレーション | 5 | 295 | 0 | イントロ/エンディング共通 |
| タイトル | 5 | 295 | 0 | イントロのみ（読み上げ） |
| イッチ | 5 | -300 | -830 | 本編 |
| スレ民1 | 5 | -300 | -830 | 本編 |
