### 2. 分析フェーズ（3エージェント並列）

**Agent tool で以下を同時起動：**

#### Agent A: 自チャンネル分析官
アナリティクス.mdと動画管理リスト.mdを精査し、以下を分析：
- 動画別パフォーマンス（再生数/CTR/視聴時間/登録者増の順位）
- 伸びた動画 vs 伸びなかった動画の共通点
- タイトル・サムネの効果（変更前後の比較があれば）
- 視聴者の傾向（コメント内容・高評価率）
- 投稿頻度と再生数の相関
→ JSON形式で分析結果を返す

#### Agent B: 競合スカウト
競合チャンネルリスト.mdの上位5チャンネルについて、WebSearchで最新動画を検索：
- 直近1ヶ月の新着動画タイトル・再生数
- バズ動画（10万再生以上）の共通パターン
- 新しく参入したチャンネルの有無
→ JSON形式で分析結果を返す

#### Agent C: トレンドウォッチャー
WebSearchで以下を調査：
- 「ガルちゃん まとめ YouTube」直近のトレンド
- X（Twitter）で「ガルちゃんまとめ」関連のバズ
- 40代女性向けYouTubeの新しいフォーマット・手法
→ JSON形式で分析結果を返す

### 3. 統合分析（ディレクター）

3エージェントの結果を統合して以下を分析：
- **自チャンネルの強み/弱み**
- **競合との差別化ポジション（更新）**
- **バズ動画の共通パターン**
- **取り入れるべき新手法**
- **次に作るべきテーマの方向性**
