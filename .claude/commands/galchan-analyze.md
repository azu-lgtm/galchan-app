### 3. 分析フェーズ（2エージェント並列）

**Agent tool で以下を同時起動。各エージェントにディレクターの方針メモを渡す：**

#### Agent A: 自チャンネル分析官
**読み込み:** アナリティクス.md + 動画管理リスト.md + 前回の自チャンネル分析レポート
以下を**YouTubeアナリスト視点**で分析：
- 動画別パフォーマンス（再生数/CTR/視聴時間/登録者増の順位）
- **視聴維持率の分析**（どの動画が何%まで維持できているか。維持率が高い動画の共通点）
- **前回からの変化**（伸びた動画・落ちた動画。なぜ伸びた/落ちたのか仮説）
- タイトル・サムネの効果（変更前後の比較があれば）
- **視聴者の行動パターン**（コメント内容・高評価率・登録者転換率）
- 投稿頻度と再生数の相関
- **伸びている動画の特徴**（テーマ/タイトル/サムネ/投稿曜日・時間の共通点）
→ JSON形式で分析結果を返す

#### Agent B: 競合動向アナリスト（旧：競合スカウト + トレンドウォッチャー統合）
**読み込み:** 競合チャンネルリスト.md + 前回の競合分析 + 勝ちパターン統合版（Aポジション部分）

**競合チャンネルリスト.md に記載のチャンネルを対象**（「分析対象外」セクションのチャンネルは除外。登録者数順に優先度をつける）。WebSearchで以下を調査：
- 各チャンネルの直近1ヶ月の新着動画タイトル・再生数
- バズ動画（10万再生以上）の共通パターン
- 新しく参入したチャンネルの有無（「ガルちゃんまとめ YouTube」で新顔を検索）
- **前回のAポジション（未開拓領域）がまだ空いているか**
- **競合が新たに参入し始めたテーマ**（Aポジションの陳腐化チェック）
- **ガルちゃんまとめ系の最新トレンド**（WebSearchで「ガルちゃん まとめ YouTube 2026」等を検索）

> **注意:** X（Twitter）APIには直接アクセスできない。WebSearchで「site:x.com ガルちゃんまとめ」を検索するが、結果が限定的な場合がある。その場合は他のソースで補完する。

→ JSON形式で分析結果を返す

#### Agent C: コメント深掘りアナリスト（NEW）
**読み込み:** 競合チャンネルリスト.md + DB/logs/運用知見.md

**競合と自チャンネルの両方のコメントを深く分析する。**

**自チャンネルのコメント分析:**
- YouTube Data API で自チャンネル動画のコメントを取得
- 各動画のコメント傾向を分類:
  - **共感系**（「私も〜」「わかる」）→ 刺さったポイント
  - **驚き系**（「知らなかった」「マジで？」）→ 新情報として価値があったポイント
  - **要望系**（「次は○○やって」「○○も知りたい」）→ ネタヒント
  - **不満系**（「違うと思う」「長い」）→ 改善ポイント
  - **拡散系**（「友達にも教える」「シェアした」）→ バイラル要素
- **なぜその動画が伸びたか/伸びなかったか**をコメント内容から分析

**競合のコメント分析:**
- 競合のバズ動画（10万再生超）のコメント上位20件をWebFetchで取得
- 視聴者が何に反応しているかを分類（上記と同じ5分類）
- **自チャンネルにない視聴者ニーズ**を抽出
- **「次は○○」系の要望**をネタ候補として収集

**出力:**
- コメントから抽出したネタヒント一覧（DB/logs/ネタ候補.mdに追記候補として）
- 自チャンネルの改善ポイント
- 競合との視聴者反応の違い
→ JSON形式で分析結果を返す

### 3.5. 競合バズ動画の文字起こし自動取得（NEW）

> Agent Bで新しいバズ動画（10万再生超）を検出した場合、自動で文字起こしを取得する。

**手順:**
1. Agent Bの結果から、前回分析以降に新たに検出されたバズ動画のURLを抽出
2. 既存の `競合分析/台本文字起こし/` フォルダ内のファイル名と照合 → 未取得の動画のみ対象
3. 対象動画のYouTube字幕（自動生成）をWebFetchで取得
   - URL: `https://www.youtube.com/watch?v={videoId}` からWebFetchで字幕テキストを抽出
   - 字幕が無い場合はスキップし、レポートに「字幕なし」と記載
4. 取得した文字起こしを整形して保存:
   - 保存先: `C:\Users\meiek\Dropbox\アプリ\remotely-save\obsidian\02_youtube\ガルちゃんねる\競合分析\台本文字起こし\`
   - ファイル名: `{動画タイトル短縮} 文字起こし.md`
5. 取得結果をレポートに記載（取得数・スキップ数）

**制限:**
- 1回の分析で最大5本まで（多すぎると時間がかかる）
- 文字起こし取得に失敗した場合はスキップして続行
- ユーザーが「この動画の文字起こし取って」と指示した場合は制限なしで取得
