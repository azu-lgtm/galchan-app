# /galchan-detector — 異常検知レポート

YouTube Analytics API から前週比データを取得し、異常を検知してレポートを出力する。

## 手順

### 1. devサーバー確認

- `localhost:3001` が起動しているか確認する
- 起動していなければ `npm run dev` で起動（バックグラウンド）

### 2. Detector 実行

`GET http://localhost:3001/api/youtube/detector?save=true` を実行。

### 3. 結果の解釈とレポート

APIレスポンスの内容をユーザーにわかりやすく報告する。

**報告フォーマット:**

```
## Detector レポート（YYYY-MM-DD）

### サマリー
（summary の内容）

### チャンネル前週比
| 指標 | 先週 | 今週 | 変化率 |
|---|---|---|---|
| 再生数 | ... | ... | ...% |
| 視聴時間 | ... | ... | ...% |
| 登録者純増 | ... | ... | ...% |

### アラート
（alerts の内容。なければ「異常なし」）

### 動画別アラート
（videoAlerts の内容。なければ「異常なし」）

### 推奨アクション
（アラートの内容に基づいて、具体的な次のアクションを1-3個提案する）
- 例: CTRが低い → サムネ・タイトルの見直しを提案
- 例: 維持率が低い → 台本構成の見直しを提案
- 例: 再生数減 → 投稿頻度やテーマの見直しを提案
```

### 4. Obsidian保存確認

`save=true` で実行しているので、以下に自動保存される:
`02_youtube/ガルちゃんねる/自分動画/Detectorレポート.md`

保存完了をユーザーに報告する。
