### 2. リサーチフェーズ（3エージェント並列）

**Agent tool で以下を同時起動：**

#### Agent A: ガルちゃんトレンドリサーチャー
WebSearchで以下を調査：
- 「ガルちゃん 買って後悔」「ガルちゃん 知らなかった」「ガルちゃん やめてよかった」等のトピック
- 直近1ヶ月でコメント数が多いガルちゃんトピック
- 40代女性が反応している話題
→ ネタ候補10本をJSON形式で返す

#### Agent B: 競合ギャップアナリスト
競合チャンネルリスト.mdの上位チャンネルの動画タイトルをWebSearchで取得し：
- 競合が扱っていないテーマ（空白地帯）
- 競合がバズらせているが自チャンネルの「失敗回避」軸で差別化できるテーマ
- 競合の低再生動画（避けるべきテーマ）
→ ギャップ分析結果をJSON形式で返す

#### Agent C: 公的機関ネタスキャナー
WebSearchで以下を調査：
- 国民生活センター・消費者庁・農林水産省の最新注意喚起
- 厚生労働省の健康関連の新しい発表
- 40代女性に関係する公的機関の情報
→ 公的データに基づくネタ候補5本をJSON形式で返す

### 3. ネタスコアリング（ディレクター）

3エージェントの結果を統合し、以下の基準でスコアリング：

| 基準 | 配点 | 説明 |
|---|---|---|
| 勝ちパターン合致 | 30点 | 感情ワード×権威性×40代訴求 |
| 競合未開拓度 | 25点 | 空白地帯か、差別化できるか |
| ガルちゃん共感度 | 20点 | 実際のコメント/票数 |
| 公的データ裏付け | 15点 | 具体的な数値・事実あり |
| 既存ネタとの重複なし | 10点 | 過去台本と被らない |

### 4. 出力形式

15〜20本のネタを以下の形式で生成：

```
## テーマ案
| # | テーマ | 想定タイトル案 | スコア | 優先度 | メモ |
|---|---|---|---|---|---|
| 1 | （テーマ） | （タイトル案） | 85/100 | ★★★ | （根拠・競合状況） |
...
```

優先度基準：
- ★★★（80点以上）競合未開拓 × 勝ちパターン合致
- ★★（60〜79点）競合済みだが差別化できる
- ★（59点以下）アイデアとして保存
