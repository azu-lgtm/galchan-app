# /galchan-post-tracker — 投稿後トラッキング

最新の投稿動画のパフォーマンスを追跡する。投稿からの経過時間で自動的にチェック内容が変わる。

- **48時間未満（Stage 1: 初動）**: 再生数・いいね・コメント
- **48時間以降（Stage 2: フル判定）**: CTR・維持率・インプレッション + チャンネル平均との比較

## 手順

### 1. devサーバー確認

- `localhost:3001` が起動しているか確認する
- 起動していなければ `npm run dev` で起動（バックグラウンド）

### 2. トラッキング実行

**最新動画を自動検出:**
`GET http://localhost:3001/api/youtube/post-tracker?save=true`

**特定の動画を指定する場合:**
`GET http://localhost:3001/api/youtube/post-tracker?id=VIDEO_ID&save=true`

### 3. 結果の解釈とレポート

**Stage 1（初動）の場合:**
- 再生数の勢いを評価
- サムネ・タイトル変更の判断材料を提示
- 「あと○時間後にフル判定が可能」を伝える

**Stage 2（フル判定）の場合:**
- CTR・維持率・インプレッションを評価
- チャンネル平均との比較
- 成功/失敗の判定 → Playbook に記録すべきポイントを提案

### 4. 推奨アクション

アラートに基づいて具体的なアクションを1-3個提案する:
- CTRが低い → サムネ・タイトルの差し替え案を提示
- 維持率が低い → 次の台本で冒頭/中盤の構成改善を提案
- 再生数が低い → テーマ選びの振り返りを促す
