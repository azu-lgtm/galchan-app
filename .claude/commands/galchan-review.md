# /galchan-review — 台本品質チェック

生成済みの台本をルールに照らし合わせてレビューし、修正指示を出します。

## 引数
- `$ARGUMENTS` = 台本ファイルパス（TSV or MD）
- 引数なし: `台本/` フォルダ内の最新ファイルを自動選択

---

## 手順

### 1. 対象ファイルの読み込み

台本ファイル（TSV or MD）を読み込む。

### 2. ルールファイルの読み込み

以下を必ず読む：

- `C:\Users\meiek\Dropbox\アプリ\remotely-save\obsidian\02_youtube\ガルちゃんねる\分析結果\勝ちパターン.md`

### 3. done台本との照合

- `C:\Users\meiek\Desktop\ClaudeCode-projects\galchan-app\local-tool\tsv_input\国民生活センター注意喚起商品_done.tsv`

スクリプトが存在しない場合はスキップして手動チェックに進む。

### 5. チェック項目（全項目を1つずつ確認して結果を報告）

#### A. フォーマットチェック（最重要）
- [ ] 1行目が「ナレーション」で始まっているか
- [ ] 2行目が「タイトル」で、10〜15文字程度のテーマか
- [ ] ヘッダー行がないか（話者/本文/SE等のヘッダーは禁止）
- [ ] コロン（：）が使われていないか
- [ ] 登場人物名がルール通りか（ナレーション/イッチ/スレ民1〜6のみ）
- [ ] 「ナレーター」「イチコ」等の旧名が使われていないか
- [ ] シーン転換行がないか
- [ ] 「第1話」「エピソード1」等の番号マーカーがないか

#### B. 構造チェック
- [ ] 総行数が285〜350行の範囲内か（done.tsv=292行が基準）
- [ ] イントロが5〜7行か（ナレーション+タイトル）
- [ ] 各トピックが45〜55行か
- [ ] エンディングが10〜12行か（ナレーション）
- [ ] ナレーションが冒頭とエンディングにしか出現しないか

#### C. 会話ルールチェック
- [ ] 同じ話者が2回連続で発言していないか（イントロ・エンディングのナレーション除く）
- [ ] イッチの発言が多すぎないか（全体の15〜20%程度が目安）
- [ ] スレ民1〜6が全員登場しているか
- [ ] スレ民の発言に偏りがないか

#### D. 重複チェック
- [ ] 「怖い」「知らなかった」「ありがとう」が1トピック内で3回以上ないか
- [ ] 同じ驚き表現（「えっ」「ウソでしょ」「マジで」等）がトピック間で繰り返されていないか
- [ ] 同じ感情パターンの発言順が2トピック以上で繰り返されていないか

#### E. 感情導線チェック
- [ ] 6トピックで最低4種類の感情が使われているか
- [ ] 同じ感情トーンが2話連続していないか
- [ ] トピック内の流れが「失敗提起→共感→深掘り→解決策→まとめ」になっているか

#### F. コンテンツ品質チェック
- [ ] イッチの発言に具体的な数値・機関名があるか（全トピックで確認）
- [ ] 「〜らしい」「〜かもしれない」の曖昧表現がないか
- [ ] 対策が具体的か（「気をつける」ではなく「○℃以下で保存」等）
- [ ] 権威機関名が繰り返されすぎていないか（最初に1回で十分）
- [ ] 専門用語が多すぎないか（ガルちゃんの口調を維持）

#### G. SEチェック
- [ ] SE配置が10行に1回程度か（許容: 8〜12行間隔）
- [ ] SE1/SE2が交互に配置されているか
- [ ] ナレーション・タイトル行にSEがないか

#### H. タイトル・サムネチェック
- [ ] タイトルに感情ワードが前半に入っているか
- [ ] 「40代」直接訴求が含まれているか
- [ ] サムネ上段と下段で情報が被っていないか
- [ ] サムネとタイトルで同じことを言っていないか

### 6. 出力フォーマット

```markdown
## 台本レビュー結果

### 対象: 【自ガルN台本】タイトル
### 判定: ✅合格 / ⚠️要修正 / ❌不合格

---

### 合格項目
- ✅ （合格した項目を列挙）

### 要修正項目
#### 🔴 重大（修正必須）
- 問題: （具体的な問題）
- 箇所: （行番号）
- 修正案: （具体的な修正テキスト）

#### 🟡 軽微（改善推奨）
- 問題: （具体的な問題）
- 箇所: （行番号）
- 修正案: （具体的な修正テキスト）

### スコア
| 項目 | 点数 |
|---|---|
| フォーマット | /15 |
| 構造 | /10 |
| 会話ルール | /15 |
| 重複なし | /15 |
| 感情導線 | /15 |
| コンテンツ品質 | /15 |
| SE・テンポ | /5 |
| タイトル・サムネ | /10 |
| **合計** | **/100** |

### 総評
（1〜2行で全体の評価と最優先の改善ポイント）
```

80点以上 → ✅合格（そのまま使える）
60〜79点 → ⚠️要修正（修正指示に従って部分修正）
59点以下 → ❌不合格（全面再生成を推奨）
