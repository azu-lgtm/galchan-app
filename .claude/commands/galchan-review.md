### 4. チェック項目（全項目を1つずつ確認して結果を報告）

#### A. フォーマットチェック（台本生成ルール.md準拠）
- [ ] 1行目が「ナレーション」で始まっているか
- [ ] 2行目が「タイトル」で、10〜15文字程度のテーマか
- [ ] ヘッダー行がないか（話者/本文/SE等のヘッダーは禁止）
- [ ] コロン（：）が使われていないか
- [ ] かぎ括弧「」が使われていないか
- [ ] 登場人物名がルール通りか（ナレーション/タイトル/イッチ/スレ民1〜6のみ）
- [ ] シーン転換行がないか
- [ ] 「第1話」「エピソード1」等の番号マーカーがないか

#### B. 構造チェック（勝ちパターン統合版 + 競合台本構造分析 準拠）
- [ ] テーマタイプは何か（注意喚起/商品/マナー/健康美容）
- [ ] トピック数がテーマタイプ別の推奨範囲内か
  - 注意喚起系: 12-17 / 商品系: 25-35 / マナー系: 15-20 / 健康美容系: 10-15
- [ ] 総行数が285〜350行の範囲内か
- [ ] 本文文字数が9,000〜10,000字か
- [ ] 1行が30字以上か（基本35字、たまに70字）
- [ ] イントロが4-5行か（10秒以内）
- [ ] エンディングが8-10行か（ナレーション）
- [ ] ナレーションが冒頭とエンディングにしか出現しないか
- [ ] **後半のテンポが前半より速いか**（後半のトピックが前半より短いか）
- [ ] **感情ピークが前半（最初の1/3）に配置されているか**

#### C. トーンチェック（競合文字起こしとの比較）★最重要
- [ ] **説明的になっていないか**（「〇〇には△△が含まれており」型の発言がないか）
- [ ] **体験談が主役になっているか**（情報提供ではなく体験の共有）
- [ ] **「教科書を読み上げる先生」ではなく「ママ友の井戸端会議」のトーンか**
- [ ] 自問・葛藤型の発言があるか（「なんで買ったんだろう」等）
- [ ] 共感の連鎖があるか（「うちも〜」→別の体験→さらに共感）
- [ ] 短い感情表現があるか（「無駄にでかくて邪魔。」等）
- [ ] 専門用語が少ないか（40代主婦の日常会話トーン）
- [ ] 整いすぎた会話になっていないか（少し未整理な温度を残す）

#### D. 会話ルールチェック（台本構成分析 準拠）
- [ ] 同じ話者が2回連続で発言していないか（イントロ・エンディングのナレーション除く）
- [ ] イッチが**冒頭1回のみ**か（本文中に出ていないか）
- [ ] スレ民1〜6が全員登場しているか
- [ ] スレ民の発言に偏りがないか
- [ ] スレ民ごとに口調・温度・視点が少しずつ違うか

#### E. トピック間チェック
- [ ] **トピック間がぶつ切りか**（接続語「あと」「そういえば」「もうひとつ」がないか）
- [ ] 前の話題の最後の発言と次の話題の最初の発言が自然に切り替わっているか
- [ ] 「あと〇〇」型の列挙感がないか

#### F. 重複チェック
- [ ] 「怖い」「知らなかった」「ありがとう」が1トピック内で2回以内か
- [ ] 同じ驚き表現がトピック間で繰り返されていないか
- [ ] 同じ感情パターンの発言順が2トピック以上で繰り返されていないか
- [ ] 感嘆符（！）の連続使用がないか（2文連続NG）

#### G. 感情動線チェック（台本構成分析 準拠）
- [ ] 最低4種類の感情が使われているか
- [ ] 同じ感情トーンが2トピック連続していないか
- [ ] 中盤に軽い話題（トーン変化）が1つ以上あるか
- [ ] 重い話の後に「少し笑える毒」や「温度差」があるか

#### H. 事実確認チェック（公的機関引用）
- [ ] 公的機関名が台本全体で2-3回以内か
- [ ] 公的機関の引用が**検証済みデータのみ**か（捏造・推測がないか）
- [ ] 「〇〇が警告している」→ 実際の警告文書があるか
- [ ] 数値が公式発表のままか（丸め・誇張がないか）
- [ ] 断定表現がないか（「がんになる」→「リスクが指摘されている」）

#### I. SEチェック
- [ ] SE配置が8〜12発言ごとか
- [ ] SE1/SE2が交互に配置されているか
- [ ] ナレーション・タイトル行にSEがないか

#### J. エンディングチェック
- [ ] **コメント誘導が入っているか**（具体的な問いかけ形式）
- [ ] まとめ→問いかけが自然につながっているか
- [ ] 固定文が一字一句正確か
- [ ] 前向きなまとめに具体的行動が含まれているか
